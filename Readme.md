# News Cover API 

## Getting Start

### Configure NodeJS
Use version 18 lts

``` bash
nvm install --lts
nvm use --lts
```

## Install dependencies
Install the following NodeJS packages

```bash
npm install express cors body-parser nodemon mongoose
```
